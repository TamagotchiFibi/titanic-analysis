# Анализ выживаемости пассажиров Титаника

## О проекте
Первый учебный проект по анализу данных на основе знаменитого датасета Титаник. Цель - выяснить, какие факторы влияли на выживаемость пассажиров при кораблекрушении.

## Данные
- **Источник:** [Kaggle Titanic Competition](https://www.kaggle.com/competitions/titanic)
- **Объём:** 891 пассажир, 12 признаков
- **Пропуски:** обработаны - Age заполнен медианой, Cabin удалён (77% пропусков), Embarked заполнен модой

## Что исследовалось
- Выживаемость по полу и классу каюты
- Возрастной профиль пассажиров
- Топ-5 самых молодых выживших
- Ценовая политика билетов по классу и полу

## Ключевые выводы
- Женщины 1 класса выживали в **96.8%** случаев, мужчины 3 класса - лишь в **13.5%**
- Все 5 самых молодых выживших - дети до года, причём из 2 и 3 классов
- Медианная цена билета женщин вдвое выше мужской - отражение социальных норм начала XX века
